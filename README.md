# 🗺️ Mappa Interattiva del Paese

## Cosa fa il progetto
Questo progetto è una **mappa interattiva** di un paese che mostra i principali punti di interesse.

All’interno della mappa è possibile trovare:
- **Locali** (bar, ristoranti, pub)
- **Farmacie**
- **Attività commerciali**
- **Altri servizi utili**

L’obiettivo è rendere **semplice e veloce** trovare ciò che serve direttamente sulla mappa.

---

## Perché il progetto è utile
La mappa è utile perché raccoglie in un unico posto informazioni che altrimenti sarebbero sparse.

Può essere utilizzata sia dai **residenti** che dai **visitatori** per:
- Orientarsi meglio nel paese
- Scoprire attività e servizi
- Trovare rapidamente punti importanti come farmacie o negozi
